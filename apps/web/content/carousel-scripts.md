## Hinweise zur Produktion

- **Format:** 9:16 vertikal, 768×1376px, JPG (TikTok lehnt PNG ab)
- **Kein Text unten:** Die unteren 20% freihalten (TikTok-Controls)
- **Branding:** 1of10-Logo oben rechts, goldene Akzentfarbe (#D4AF37)
- **Font:** Bold Sans-Serif, min. 48px für Lesbarkeit auf Mobile
- **Slide-Generierung:** Gemini API (Free Tier) unter aistudio.google.com
- **Publishing:** Upload-Post API für Multi-Plattform (TikTok + Instagram gleichzeitig)
